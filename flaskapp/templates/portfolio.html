Ski Resort Website

Topic: Full Stack Web Development

Developed a website hosted on an AWS EC2 server that provides an interface for searching for and viewing ski resorts in the US. Batting Order Simulation

Topic: Open Source Data Analysis

Created a Python program to determine the best batting order for a given lineup of MLB players. The user may specify the lineup, how many games each order will be simulated for, and what plate appearance outcomes each player had during a game. The program is object oriented in nature, which supplies users an easy API to run their own simulations. The project was published in PyPI, so that it can be pip-installable.

Topic: Data Engineering
